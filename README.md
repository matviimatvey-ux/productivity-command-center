#Productivity Command Center (PCC)

Modern JavaScript Single Page Application for productivity management.

#Overview

Productivity Command Center is a frontend SPA designed to simulate a real-world productivity system including:

- Kanban Task Board (Backlog / Doing / Done)
- Pomodoro Timer
- Habit Tracker with Streak Logic
- Smart Analytics Dashboard
- LocalStorage persistence (offline-first)

This project demonstrates frontend architecture, state management, UI logic separation and advanced DOM manipulation.

---

#Tech Stack

- HTML5
- CSS3 (Grid + Flexbox)
- Vanilla JavaScript (ES6+)
- LocalStorage API
- Modular state-driven architecture

---

#Architecture Highlights

- Centralized App State
- Dynamic Rendering
- Component-like UI sections
- Event-driven updates
- Persistent storage logic
- Statistics auto-calculation

---

#Features

#ask Management
- Create tasks
- Move between columns
- Track active vs completed tasks
- Completion rate auto calculation

#Pomodoro
- Start / Pause / Reset
- Custom session presets
- Daily session tracking

#Habit Tracking
- Streak counter
- Best streak logic
- Daily completion tracking

#Analytics
- Tasks done
- Active tasks
- Completion rate
- Pomodoro sessions
- Habit streak tracking

---

#Offline First

All data is stored in browser LocalStorage.
No backend required.
